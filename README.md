### About me

Hey! I work in quality engineering — currently at **[eGov Foundation](https://egov.org.in/)**, and a long-time contributor at **[Open Healthcare Network](https://github.com/ohcnetwork)** where I've spent the last 5+ years helping ship a Digital Public Good used to run TeleICU and decentralised healthcare capacity across states.

My days are spent breaking things on purpose — writing end-to-end tests, chasing regressions, and building test infrastructure so the people who depend on this software can trust it.

Off the keyboard, I'm usually on my bike chasing a new route or a good chai.

- Currently focused on **Playwright** automation for [`ohcnetwork/care_fe`](https://github.com/ohcnetwork/care_fe)
- Comfortable across **Python** and **TypeScript** test stacks (Pytest, Selenium, Playwright)
- Interested in test data management, CI performance, and developer experience for QA
- Based in **India** · Remote-first
